ハイアルチ PLダッシュボード｜新店追加マニュアル

2026-05 版 / A4 1枚 / 配布用
新しい店舗のオープン・店長交代時の初回セットアップ手順と、 毎月の月次データアップロード手順をまとめています。 迷ったら松尾さん（admin）に連絡してください。

ログインアカウントの発行（初回のみ）

  1. 松尾さんに以下を伝える: 店舗名 / 店長氏名 / メールアドレス
  2. 松尾さんが「設定 → ユーザー管理」でアカウントを作成 admin作業
  3. 発行されたユーザー名 / パスワードでログイン → 自店舗のみ操作可能

ハコモノから月次CSVをダウンロード（毎月1日〜）

ハコモノにログイン → レポート出力画面から下記4種類をダウンロードします。

CSV種別 ハコモノ画面 ファイル名例 必須
ML001 会員一覧 会員管理 → 会員エクスポート ML001_xxx.csv 必須
MA002 月次サマリ レポート → 月次集計 MA002_xxx.csv 必須
PL001 売上明細 レポート → 売上明細 PL001_xxx.csv 必須
PS001 商品別売上 レポート → 商品別売上 PS001_2026-05.csv 必須
アンケート回答 体験／入会アンケート → エクスポート enquete_answer_xxx.csv 任意
⚠️ PS001 だけ要注意：ファイル内に年月情報が無いため、 ファイル名に 2026-05 または 202605 を必ず含めてください。

ダッシュボードにアップロード

  1. ダッシュボードにログイン → 上部メニュー「アップロード」を開く
  2. 「ハコモノ一括」タブを選択（既定で開いています）
  3. ダウンロードしたCSV を全てまとめてドラッグ&ドロップ
    • ファイル種別は自動判別（ML001 / PL001 / MA002 / PS001 / アンケート）
    • 対象店舗は自店舗が自動選択されます
  4. 「取り込む」ボタンをクリック
  5. 既にデータがある月は「上書き確認」が出るので、内容を確認して「すべて上書きして取込」を選択
🚫 空ファイル / 中身が違うCSV は絶対にアップロードしない： その店舗・月のデータが消える可能性があります。心配な場合は松尾さんに確認。

ダッシュボードで反映確認

上部メニュー「ダッシュボード」を開いて、当月のデータが反映されているか確認します。

✓ 売上セクション

  • 会費売上 / パーソナル / 物販 / その他
  • 0円 や 前月と極端に違う場合は再確認

✓ 会員セクション

  • 在籍 / 新規入会 / 退会 / 体験者数
  • 新規入会率（=新規÷体験）が表示される
